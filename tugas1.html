CSS adalah salah satu bahasa pemrograman desain web (style sheet languages) yang mengontrol format tampilan sebuah halaman web yang ditulis dengan bahasa penanda (markup language) Biasanya CSS digunakan untuk mendesain sebuah halaman HTML dan XHTML, tetapi sekarang bahasa pemrograman CSS bisa diaplikasikan untuk segala dokumen XML, termasuk SVG dan XUL.
